## 前言

该项目是为了帮助开发人员快速的搭建基于vue框架开发的后台类应用项目和作为一个后台类应用实例提供给vue学习者。
项目内已写好后台类应用的基本功能和集成一些其他功能提供选择以及实例作为使用参考等。
