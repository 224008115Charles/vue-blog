## 前言

该项目是为了帮助开发人员快速的搭建基于vue框架开发的后台类应用项目和作为一个后台应用实例提供给vue学习者参考学习。
项目内已经写好后台类应用的基本功能和集成一些功能提供选择以及一些使用实例作为参考等。
